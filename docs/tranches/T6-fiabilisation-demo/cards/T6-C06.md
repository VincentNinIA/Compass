## 5. Contrats, types, entrées et sorties

URL HTTPS, permissions micro cohérentes, responsive au viewport cible, parcours clavier, contrastes/status, mentions/licence visibles.

## 6. Séquence d'implémentation décidée

Le build production est servi par `scripts/serve-https.mjs`, qui exige certificat
et clé hors du bundle. Le candidat envoie les permissions microphone/camera
same-origin, puis le scénario navigateur contrôle axe, clavier, reduced motion,
viewports, modes dégradés et attributions. Le runbook jury est
`docs/DEMO_RUNBOOK.md`.

## 7. Cas d'erreur et comportements de repli

Permission ou service externe indisponible: mode explicite et chemin typed/scripted; aucun secret dans HTML.

## 8. Critères d'acceptation Given/When/Then

Given URL jury sur machine propre, When golden journey est parcouru clavier/souris, Then contenu reste utilisable et attribution présente.

Verdict : `pass` sur un contexte navigateur neuf et le vrai applet. Le candidat
local HTTPS est `https://127.0.0.1:3443`, avec certificat éphémère uniquement
pour le harness; un certificat de confiance reste requis sur la machine jury.
