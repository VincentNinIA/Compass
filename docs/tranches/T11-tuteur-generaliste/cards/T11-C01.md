# T11-C01 — Tuteur généraliste sûr

## Statut

`done — pass`

## Intention

Permettre à un élève de présenter n'importe quel exercice scolaire lisible sans
être bloqué par une liste de matières. Compass confirme d'abord ce qu'il a lu,
puis accompagne par la voix ou le texte sans prétendre disposer d'un outil de
vérification lorsqu'aucun module spécialisé ne correspond.

## Contrats

- `GeneralExerciseWireV1` est un objet Structured Outputs fermé et borné.
- Issues autorisées : `ready` et `needs_clarification`; aucun `unsupported` de
  matière n'existe dans ce contrat.
- Le client revalide l'enveloppe et n'active le coach qu'après un clic de
  confirmation unique.
- Le texte de l'exercice est une donnée non fiable : il n'entre jamais dans les
  instructions système, ne déclenche aucun outil et ne produit aucune preuve.
- Le profil Realtime `general_tutor` possède `tools:[]` et
  `tool_choice:"none"`. L'item de contexte est créé sans `response.create`.

## Acceptation

- Given la capture française de géométrie à six questions, When la route réelle
  l'analyse, Then elle retourne `ready` avec six tâches ordonnées.
- Given un exercice lisible d'une autre matière, When il est analysé, Then la
  matière ne provoque jamais `unsupported`.
- Given une photo illisible, When elle est analysée, Then une question de
  clarification bornée est rendue et aucun contexte coach n'est émis.
- Given un exercice confirmé, When le coach voix ou texte s'ouvre, Then son
  contexte est envoyé une seule fois comme item utilisateur et ses outils sont
  vides.
- Given le parcours général, When la page est inspectée, Then aucun canevas ou
  libellé médiatrice/AB/PA-PB n'est affiché et le reflow reste valide.

## Vérification attendue

```sh
pnpm --dir apps/frontend lint
pnpm --dir apps/frontend typecheck
pnpm --dir apps/frontend test --run
pnpm --dir apps/frontend build
pnpm --dir apps/frontend exec playwright test --grep-invert @live
```

## Preuves

- Lecture credentialed de la capture utilisateur : HTTP 200,
  `ready_general`, six tâches françaises dans l'ordre.
- Contrats génériques : mathématiques, histoire, langue étrangère, répétitions
  et clarification d'image illisible couvertes sans issue `unsupported`.
- Realtime : profils voix et texte sans outil; contexte confirmé envoyé une
  seule fois comme donnée utilisateur et sans réponse automatique.
- Gates : lint, typecheck et build passent; Vitest 629/629 sur 54 fichiers;
  Playwright hors live 34/34.
- Preuve responsive : `output/playwright/T11-general-exercise-390x844.png`,
  EN/FR, six étapes et aucun débordement horizontal.
