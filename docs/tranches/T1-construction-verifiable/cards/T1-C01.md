# T1-C01 — Adaptateur GeoGebra

## 1. Résultat attendu

Encapsuler une version GeoGebra validée derrière une API TypeScript à cycle `idle→loading→ready→disposed` et cleanup idempotent.

## 2. Valeur visible pour l'utilisateur ou le jury

La construction devient observable, explicable et vérifiable sans dépendre d'un modèle.

## 3. Prérequis et dépendances

Dépendances : T0-C06. Les preuves de ces cartes doivent être disponibles.

## 4. Périmètre inclus

Encapsuler une version GeoGebra validée derrière une API TypeScript à cycle `idle→loading→ready→disposed` et cleanup idempotent. Sont inclus les types, le cycle de vie, les erreurs et le cleanup nécessaires.

## 5. Contrats, types, entrées et sorties

`load`, `withApi`, registre central des listeners et `dispose`; tout appel hors état autorisé renvoie une erreur discriminée.

## 6. Séquence d'implémentation décidée

Définir types; pinner version; encapsuler `appletOnLoad`; centraliser inscriptions; tester doubles load/dispose.

## 7. Cas d'erreur et comportements de repli

callback tardif ou méthode absente: ignorer l'epoch périmée, libérer les ressources et exposer une erreur typée.

## 8. Critères d'acceptation Given/When/Then

Given chaque état du cycle, When une méthode est appelée, Then seules les transitions autorisées accèdent à l'API et deux dispose valent un.

## 9. Tests et commandes de vérification

À exécuter : tests machine d'état, load/unmount et comptage listeners. Puis `pnpm lint`, `pnpm typecheck`, `pnpm test` et, si UI touchée, `pnpm build`. Aucun résultat n'est encore revendiqué.

## 10. Preuves à consigner avant clôture

Version chargée, matrice de transitions, listeners avant/après et sorties de tests.

## 11. Documentation à consulter

[Apps API](https://geogebra.github.io/docs/reference/en/GeoGebra_Apps_API/), « Obtaining the API Object ». [Applet Parameters](https://geogebra.github.io/docs/reference/en/GeoGebra_App_Parameters/), paramètres `appName` et `appletOnLoad`.

## 12. Hors périmètre

Validation métier et UI.

## 13. Risques et mitigations

Couplage à l'API globale; limiter la façade aux méthodes requises.
